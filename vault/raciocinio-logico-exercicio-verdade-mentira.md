## Objective

Compreender o conceito de raciocínio lógico e aprofundar a análise de um problema envolvendo afirmações verdadeiras e falsas, aplicando testes de consistência para identificar o culpado.

## Conversation

Usuário: Raciocínio lógico  
Assistente: Explicação sobre raciocínio lógico, incluindo tipos (dedutivo, indutivo e abdutivo), elementos e aplicações.

Usuário: Continue  
Assistente: Apresentação de exercícios práticos, incluindo o exercício de verdade e mentira com três pessoas.

Usuário: Explique o exercício 2  
Assistente: Explicação passo a passo testando cada hipótese (A, B ou C dizendo a verdade).

Usuário: Aprofunde a explicação  
Assistente: Análise aprofundada do problema:

Enunciado:
- A diz: "B é culpado."
- B diz: "C é culpado."
- C diz: "Eu não sou culpado."
- Apenas uma afirmação é verdadeira.

Análise:

Caso 1: A diz a verdade  
→ B é culpado  
→ B mente: C não é culpado  
→ C mente: ele é culpado  
→ Contradição (dois culpados) → inválido

Caso 2: B diz a verdade  
→ C é culpado  
→ A mente: B não é culpado  
→ C mente: ele é culpado  
→ Consistente → válido

Caso 3: C diz a verdade  
→ C não é culpado  
→ A mente: B não é culpado  
→ B mente: C não é culpado  
→ Ninguém é culpado → inválido

Conclusão:
A única solução lógica é que B diz a verdade e C é o culpado.

## Conclusions & Deliverables

- Apenas uma hipótese satisfaz todas as condições.
- Testar cada possibilidade é essencial em lógica.
- O culpado é C.
- B é o único que diz a verdade.
